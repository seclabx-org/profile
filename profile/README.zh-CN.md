# SeclabX · Secure Laboratory X

SeclabX 是一个聚焦于 **网络安全教育、实践训练与应用研究** 的开放型实验室平台。

我们围绕 **教育教学、CTF 实训、安全实验靶场、科研论文源码与开源工具** 等方向，构建教—学—研—用一体化的安全技术生态，致力于打通 **教育、实践与研究** 之间的壁垒。

---

## 📂 仓库结构说明

SeclabX 采用统一的仓库前缀体系，用于区分不同类型的项目与资源：

- **`seclabx-web-*`**  
  平台与网站类项目，包括门户网站与 CTF 平台等系统。

- **`seclabx-edu-*`**  
  教育与教学资源类项目，包括网络安全科普与课程思政相关内容。

- **`seclabx-lab-*`**  
  实验与实训类项目，主要用于课程实验、CTF 题库与实战训练。

- **`seclabx-research-*`**  
  科研与论文相关项目，用于发布论文实验代码与研究实现。

- **`seclabx-tool-*`**  
  工具与支撑类项目，包括安全工具、脚本与实验性程序。
